WEB DEVELOPMENT SKILLS

Learn HTML

HTML stands for Hyper Text Markup Language. HTML is the content of a web page. Examples of HTML include headings, paragraphs, images, links, lists, etc.

Check out the HTML tutorial on w3schools

Learn CSS

CSS stands for Cascading Style Sheets. CSS is the style of a web page. Examples of CSS include font styles and color, background color, borders, dimensions, layout, etc.

Check out the CSS tutorial on w3schools

Learn Javascript

Javascript is the programming language of the web JavaScript helps to make web pages interactive and dynamic. Some common Javascript tasks are:

  • add new HTML to the page
  • change the existing content/style
  • react to user events (mouse clicks, key presses, etc.)

Check out the Javascript tutorial on javascript.info
